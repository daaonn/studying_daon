***<문제>***


USED_GOODS_BOARD와 USED_GOODS_REPLY 테이블에서 2022년 10월에 작성된 게시글 제목, 게시글 ID, 댓글 ID, 댓글 작성자 ID, 댓글 내용, 댓글 작성일을 조회하는 SQL문을 작성해주세요. 결과는 댓글 작성일을 기준으로 오름차순 정렬해주시고, 댓글 작성일이 같다면 게시글 제목을 기준으로 오름차순 정렬해주세요.


-- 오답1
```sql
SELECT ub.TITLE, ub.BOARD_ID, ur.REPLY_ID, ur.WRITER_ID, ur.CONTENTS, ur.CREATED_DATE
From Used_Goods_Board ub
join USED_GOODS_REPLY ur
on ub.BOARD_ID = ur.BOARD_ID
where year(ur.Created_date) = 2022 & month(ur.Created_date) = 10
order by ur.Created_date asc, ub.title asc;
```


논리 연산자 잘못 사용! >> AND 사용하기


-- 오답2
```sql
SELECT ub.TITLE, ub.BOARD_ID, 
    ur.REPLY_ID, ur.WRITER_ID, ur.CONTENTS, ur.CREATED_DATE
From Used_Goods_Board ub
join USED_GOODS_REPLY ur
    on ub.BOARD_ID = ur.BOARD_ID
where year(ur.Created_date) = 2022 and month(ur.Created_date) = 10
-- 조건 더 명확히 >> BETWEEN : where ur.Created_date BETWEEN '2022-10-01' AND '2022-10-31'
order by ur.Created_date asc, ub.title asc;
```


날짜 형식 설정해주기!


-- 정답
```sql
-- 코드를 입력하세요
SELECT ub.TITLE, ub.BOARD_ID, 
    ur.REPLY_ID, ur.WRITER_ID, ur.CONTENTS, 
    date_format(ur.CREATED_DATE, '%Y-%m-%d') as CREATED_DATE
From Used_Goods_Board ub
join USED_GOODS_REPLY ur
    on ub.BOARD_ID = ur.BOARD_ID
where substr(ub.created_date, 1, 7) = '2022-10'
order 
    by ur.Created_date asc, ub.title asc;
```
