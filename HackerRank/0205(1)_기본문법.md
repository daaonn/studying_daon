### 문제 1. Weather Observation Station 2
https://www.hackerrank.com/challenges/weather-observation-station-2/problem?isFullScreen=true


Query the following two values from the STATION table:

The sum of all values in LAT_N rounded to a scale of  decimal places.
The sum of all values in LONG_W rounded to a scale of  decimal places.


```SQL
-- rounded to a scale of 2
SELECT ROUND(SUM(LAT_N), 2) AS lat, ROUND(SUM(LONG_W), 2) AS lon
FROM STATION;
```
* **ROUND** : 반올림

### 문제 2. Aggregation
https://www.hackerrank.com/challenges/weather-observation-station-13/problem?isFullScreen=true


Query the sum of Northern Latitudes (LAT_N) from STATION having values greater than 38.7880 and less than 137.2345. Truncate your answer to 4 decimal places.

```SQL
-- LAT가 38.7880보다 크고 137.2345보다 작아야 함
-- 소수점 4 이후는 잘라내기
SELECT TRUNCATE(SUM(LAT_N), 4) AS LAT
FROM STATION
WHERE LAT_N > 38.7880 AND LAT_N < 137.2345;
```
* **TRUNCATE** : 숫자의 소수점 이하를 “버리는” 함수 (반올림 X)

### 문제 3. Weather Observation
https://www.hackerrank.com/challenges/weather-observation-station-14/problem?isFullScreen=true


Query the greatest value of the Northern Latitudes (LAT_N) from STATION that is less than 137.2345. 
Truncate your answer to 4 decimal places.

```SQL
-- 가장 큰 위도
-- LAT_N < 137.2345
-- 소수점 4자리까지 자르기
SELECT TRUNCATE(MAX(LAT_N), 4)
FROM STATION
WHERE LAT_N < 137.2345;
```

### 문제 4. Weather Observation Station 17
https://www.hackerrank.com/challenges/weather-observation-station-17/problem?isFullScreen=true


Query the Western Longitude (LONG_W)where the smallest Northern Latitude (LAT_N) in STATION is greater than 38.7780.
Round your answer to 4 decimal places.

```SQL
-- LONG_W을 출력
-- 가장 작은 LAT_N이 38.7780보다 큰 >> LAT_N이 38.7780보다 큰 것 중 가장 작은 거
SELECT ROUND(MIN(LONG_W), 4) AS LON
FROM STATION
WHERE LAT_N = (
  SELECT MIN(LAT_N)
  FROM STATION
  WHERE LAT_N > 38.7780);
```
* WHERE과 HAVING의 쓰임새가 아직 헷갈림

### 문제 5. Weather Observation Station 18
https://www.hackerrank.com/challenges/weather-observation-station-18/problem?isFullScreen=true


* P1(a,b) AND P2(c,d)
* a : MIN(LAT_N)
* b : MIN(LONG_W)
* c : MAX(LAT_N)
* d : MIN(LONG_W)

Query the **Manhattan Distance** between points P1 and P2 and round it to a scale of 4 decimal places.


* **맨하튼 공식** : ∣x1​−x2​∣+∣y1​−y2​∣

```SQL
-- 맨하튼 거리
-- 소수점 4까지 반올림
SELECT ROUND(ABS(MIN(LAT_N)-MAX(LAT_N)) + ABS(MIN(LONG_W)-MAX(LONG_W)), 4) AS MD
FROM STATION;
```

* **유클리드 공식** :
  <img width="167" height="50" alt="image" src="https://github.com/user-attachments/assets/85a47b46-bb9c-4166-be1e-76d4d83e3706" />

* 제곱 : POWER
* 제곱근 : SQRT

```SQL
SELECT ROUND(
  SQRT(POWER(MIN(LAT_N)-MAX(LAT_N)) + POWER(MIN(LONG_W)-MAX(LONG_W))), 4)
FROM STATION;
```​
